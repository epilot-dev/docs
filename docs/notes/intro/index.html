Notes

Notes let you attach contextual information to other entities in epilot. All notes are represented as Note entities, accessible through the Entity API and SDK.

Using Notes from the Entity sidebarâ€‹

Accessing Notes from the Entity Viewâ€‹

On any supported entity type, expand the sidebar and go to the Notes Tab to see all notes associated with that entity.

Creating a new Noteâ€‹

Click New Note in the top-right corner of the Notes Tab. You can format your note with rich text, mention other users, add file attachments, and categorize with labels.

Creating a comment on a Note threadâ€‹

Click the Comment button at the bottom of any note to start a thread. Comment threads let you discuss a topic in detail without cluttering the parent entity context.

Pinning Notesâ€‹

Click the Pin button on any note to pin it to the top of the Notes tab for quick access.

Searching Notesâ€‹

Search notes on the current entity by content, mentioned users, or labels using the search input at the top of the Notes tab.

info

You can also filter notes globally through the Note entity table. See Note Entities for details.

Note file attachmentsâ€‹

You can add file attachments to notes and comments. Attachments are stored as File entities and accessible through the File API.

Configuring Notes notificationsâ€‹

New notes automatically notify users who are:

  • Participating in the thread (created the note or replied at least once)
  • Mentioned directly in the note or comment

To change notification preferences, go to User > My Account > Notifications and find the Communication section.
